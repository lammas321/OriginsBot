## Users
The Application collects telemetry data when Interacted with for debugging and minimal usage monitoring purposes, though none of this telemetry is stored.

The telemetry the Application does collect is logged to a console window only accessible via the Owner's hardware the Application is ran from.
- The contents logged to this console are not persistently stored, meaning if the program running the Application is stopped for any reason, the telemetry is lost forever.
- So long as the program running the Application has not stopped, logs dating back to when the program started running may be visible in the console window, but not before that point.
- The Owner may open this window at any time to ensure the Application is running nominally and isn't being misused by any User.

The following telemetry may be, but isn't guaranteed to be logged when an Interaction takes place:
- The time of the Interaction,
- the type of Interaction that occurred,
- the Discord server's name and id,
- the channel's name and id,
- the User's name and id,
- and any parameters provided to the Interaction, if applicable.

Aside from this telemetry, no additional data is collected from, processed, or stored on Users who are not Creators or Admins.

As a Creator, the only piece of Personally Identifiable Information (PII) required from you is your numerical Discord user id. This is a requirement to allow you permission to provide Origin Data to the Application.

Along with other provided Origin Data, this is stored and publicly displayed upon request via the Application's Interactions.

## Origin Data
### Backups
Whenever existing Origin Data is updated, moved, or deleted via the Application's Interactions (excluding origin icon images), the existing Origin Data is backed up and timestamped to a separate backup folder only the Owner can access.
- This is done in an effort to prevent Origin Data from being lost forever by accident, due to malicious intent, or due to malfunctions.
- However, this means any Origin Data provided to the Application will remain in the backup folder until it is explicitly deleted by the Owner.

### Storage Disclaimer
Because Origin Data is intended for public display by design, and to make it easier on the Owner to maintain the Application, zero Origin Data is encrypted by the Application.
- Creators and Admins are advised that any Origin Data provided to the Application is inherently publicly visible at all times.
- The Owner takes reasonable measures to secure their hardware, but does not guarantee absolute security.

Origin Data provided to the Application that is processed and accepted by it will be persistently stored indefinitely until explicitly deleted by a Creator, Admin, or the Owner.

Origin Data that has been backed up to the backup folder will be persistently stored indefinitely until explicitly deleted by the Owner.
